Guía de despliegue

Cómo poner SkillTwin en marcha

Esta página explica los pasos para ejecutar el backend localmente, usar Docker y publicar la landing estática en GitHub Pages.

Ejecutar localmente

Arranca el dashboard en tu equipo

Desde Windows

Abre PowerShell en la raíz del repo y ejecuta:

./run.ps1

Luego ve a http://localhost:8000.

Desde Linux/macOS

Abre un terminal en la raíz del repo y ejecuta:

./run.sh

Luego ve a http://localhost:8000.

Docker

Ejecuta SkillTwin en un contenedor

Construye la imagen:

docker build -t skilltwin .

Ejecuta el contenedor:

docker run -p 8000:8000 skilltwin

Abre http://localhost:8000.

GitHub Pages

Publica la landing estática

La landing ya está preparada en docs/. Para activar GitHub Pages:

  1. Sube el repo a GitHub.
  2. En tu repositorio, abre Settings > Pages.
  3. Selecciona la rama principal y la carpeta /docs.
  4. Guarda y espera unos minutos.

La web estará disponible en https://<tu-usuario>.github.io/<tu-repositorio>/.
